단백질 구조를 넘어
분자 간 상호작용 예측으로

기존 단백질 구조 예측에 머물렀던 AI가 이제 단백질, 핵산(DNA/RNA), 리간드(약물 후보 물질) 등 거의 모든 생체 분자의 복잡한 상호작용을 단일 프레임워크 내에서 높은 정확도로 시뮬레이션할 수 있게 되었습니다.

🚀 1. 예측 대상의 획기적인 확장 및 성능 향상

AlphaFold 3는 단백질 간 결합(DockQ)을 넘어, 핵산(iLDDT), 소분자 약물, 공유결합 변형(RMSD 2Å 미만)까지 정확하게 예측해냅니다. 이는 수많은 특화 도구들을 뛰어넘는 압도적 성능입니다.

분자 타겟별 예측 정확도 비교

이전 세대 모델과 AlphaFold 3의 주요 성능 지표 비교 (높을수록 우수)

자료: Nature Extended Data Table 1 기준
약물 도킹의 혁신 (SBDD)

리간드 결합 구조 예측 시 포켓 정보를 제공하지 않은 블라인드 테스트에서도 높은 성공률을 기록하여 신약 개발 프로세스를 가속화할 잠재력을 입증했습니다.

핵산 구조 통합 예측

단백질-DNA 복합체 예측 시, 기존 핵산 특화 모델보다 더 높은 정확도를 보여 범용 딥러닝 모델의 우수성을 증명했습니다.

단백질 복합체 정확도 개선

단백질 단일 구조의 예측 정확도는 이미 정점에 달했으나, 복합체 모델링 성능은 이전 모델 대비 눈에 띄게 상승했습니다.

🧠 2. MSA 연산 축소와 확산(Diffusion) 모델의 도입

거대한 다중 서열 정렬(MSA) 데이터를 끝까지 유지하던 방식을 버리고, 쌍(Pair) 정보에 집중하는 가벼운 아키텍처로 진화했습니다. 구조 생성 시에도 기하학적 조립을 탈피하여 이미지 생성 AI와 유사한 원자 단위의 확산(Diffusion) 모듈을 적용했습니다.

🔍 3. 환각 현상 제어: 교차 증류(Cross-distillation)

확산 모델의 도입은 구조가 명확하지 않은 영역(Disordered Regions)에 그럴싸한 가짜 구조를 만들어내는 환각 부작용을 일으켰습니다. 하지만 이전 모델의 예측 결과를 교사로 삼는 기법을 통해 이 문제를 효과적으로 통제했습니다.

무질서 영역 식별 성능 (RASA AUC)

값이 1에 가까울수록 가짜 구조를 그리지 않고 올바르게 식별함을 의미합니다.

💡

데이터 분석 결과

교차 증류 학습을 적용하지 않은 확산 모델(적색)은 성능이 크게 하락했습니다. 하지만 이를 적용함으로써 이전 모델(청색) 수준의 우수한 환각 제어 성능(녹색)을 회복하는 동시에 생성형 AI의 장점만을 취할 수 있게 되었습니다.

📈 4. 생성 시드 스케일링을 통한 항체-항원 결합 탐색

항체-항원 구조처럼 유동적이고 복잡한 상호작용의 경우, 모델이 여러 개의 무작위 초기 조건(Seed)에서 수없이 많은 샘플링을 수행할수록 정확한 구조를 찾아낼 확률이 지속적으로 상승하는 흥미로운 특성을 보입니다.
